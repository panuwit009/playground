## Tech Stack

### Next.js

### Bun

---

## วิธีเปิดโปรเจค

1. ติดตั้ง runtime ที่ต้องใช้ (Node.js หรือ Bun)
2. ใช้ runtime ติดตั้ง package
3. cd ไปที่ folder project ที่ต้องการ
4. รันโปรเจคได้เลย ( npm run dev หรือ bun dev)
